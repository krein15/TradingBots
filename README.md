## Боты

### Бот #1 — EMA (Binance, 5m)

**Стратегия:** Импульс + фильтр BTC EMA50 + ML фильтр

* Сканирует 150+ монет Binance
* BTC выше EMA50 → только лонги, ниже → только шорты
* Лимитный вход по Фибо 23.6%
* Стоп: под лоу импульсной свечи
* Фильтр стопа: только 0.2–0.5% (зона максимального WR по данным)
* Блокировка плохих часов: 14:00, 21:00, 22:00 UTC (WR 7–9%)
* ML фильтр: Random Forest, порог 0.34
* Режим 24/7
* RR 1:4, риск 5%

**Параметры:**

```
timeframes    = ["5m"]       # 15m убран — WR 0% на 14 сделках
min_price_chg = 0.003
min_vol_mult  = 1.2
fibo_entry    = 0.236
min_stop_pct  = 0.002        # стоп <0.2% — слишком плотный, выбивается шумом
max_stop_pct  = 0.005        # стоп >0.5% — WR резко падает
bad_hours     = [14, 21, 22] # WR 7-9% — систематически плохие часы UTC
rr_ratio      = 4.0
risk_pct      = 0.05
session_filter = False
```

**Статистика:** 314 сделок, WR 25.5%

**Ключевые инсайты из данных (314 сделок):**
```
Стоп 0.2-0.3%               → WR 36%  (лучшая зона)
Стоп 0.2-0.5% + хор. часы  → WR 45-49%
Стоп >0.5%                  → WR ≤17% (фильтруется)
Объём 3-5x                  → WR 36%  (оптимальный диапазон)
Часы 14, 21, 22 UTC         → WR 7-9% (блокируются)
Часы 05, 10, 11, 16, 19 UTC → WR 35-50%
15m TF                      → WR 0%   (убран)
```

### Бот #2 — Структура рынка (Binance, 30m/15m)

**Стратегия:** BOS (Break of Structure) + ретест + HTF тренд + RSI дивергенция

* Свинги: 5 свечей с каждой стороны
* Структура HH+HL → бычья → лонги
* Структура LH+LL → медвежья → шорты
* **HTF фильтр:** перед лонгом тренд на 1h должен быть bull или neutral;
  перед шортом — bear или neutral. Противоположный HTF = пропуск сигнала
* **RSI дивергенция:** бычья подтверждает лонг, медвежья — шорт (логируется)
* Вход на 50% отката, стоп за свинг-хай/лой
* Режим 24/7
* RR 1:4, риск 5%

**Параметры:**

```
timeframes       = ["30m", "15m"] # 5m убран (WR 10%), 30m — чище структура
swing_bars       = 5
min_swing_pct    = 0.015
max_stop_pct     = 0.012          # стоп >1.2% — сигнал пропускается
htf_ema_period   = 50             # EMA для HTF тренда монеты
htf_neutral_zone = 0.005          # ±0.5% нейтральная зона
require_div      = False          # дивергенция — информационный фильтр
rr_ratio         = 4.0
session_filter   = False
```

**Статистика:** 89 сделок, WR 10.1% (переработан — ждём новых данных)

**Ключевые инсайты:**
```
100% лоссов — сразу стоп (цена никогда не шла к TP)
Причина: вход против HTF тренда → добавлен get_htf_trend()
15m → WR 22% vs 5m → WR 10% (5m убран)
Стоп 0.8-1.0% → WR 17% (лучший диапазон в текущих данных)
Выходные → WR 14% vs будни → WR 7%
```

### Бот #3 — SMC v2 (Bitget, 15m/1h/4h)

**Стратегия:** MSS (Market Structure Shift) + Order Block + Дивергенция

* MSS = слом структуры с объёмом 2x+
* Order Block = последняя свеча перед MSS
* Стоп за тело OB — жёсткий лимит 0.5%
* Дивергенция RSI опциональна
* HTF тренд: 15m→1h, 1h→4h, 4h→1d
* BTC тренд фильтр
* Режим 24/7
* RR 1:4, риск 5%

**Параметры:**

```
timeframes   = ["15m", "1h", "4h"]  # добавлен 15m — больше MSS сигналов
min_vol      = 2.0
mss_min_pct  = 0.005
max_stop_pct = 0.005             # стоп >0.5% — WR 5% vs 11%
require_div  = False
rr           = 4.0
interval_min = 15                # снижено с 30 — не пропускаем 15m свечи
session_filter = False
```

**Статистика:** 123 сделки, WR 8.1% (ждём новых данных после изменений)

**Ключевые инсайты:**
```
Стоп <0.5%  → WR 11%  (у всех победных сделок)
Стоп >0.5%  → WR 5%   (фильтруется)
1h TF       → WR 4%   (хуже 5m и 15m)
```

### Бот #4 — Wyckoff Spring (Bitget, 1h/4h)

**Стратегия:** Wyckoff фаза накопления + Volume Profile + Дивергенция

* SC (Selling Climax): объём 2x+, падение 2%+
* AR (Automatic Rally): отскок 1.5%+
* ST (Secondary Test): объём < 60% от SC
* Spring: ложный пробой SC лоя, закрывается обратно
* Бычья дивергенция RSI обязательна
* Дедупликация по символу (фикс повторных входов)
* Режим 24/7
* RR 1:7, риск 5%

**Параметры:**

```
timeframes     = ["1h", "4h"]
sc_vol_mult    = 2.0    # снижено с 2.5 — реалистичнее для крипты
ar_min_bounce  = 0.015  # снижено с 0.03
st_vol_ratio   = 0.6
rr_ratio       = 7.0
scan_interval  = 15     # снижено с 30
session_filter = False
```

**Статистика:** 9 сделок, WR 0% (редкая стратегия — ждём паттернов)

**Исправления:**
```
Дедупликация по символу — фикс повторных входов (HUMA/USDT входил 4 раза)
sc_vol_mult 2.5 → 2.0
ar_min_bounce 3% → 1.5%
```

## ML модель (Бот #1)

### Текущая модель

* Алгоритм: Random Forest + калибровка вероятностей
* Обучена на: 314 EMA сделках
* ROC-AUC: 0.584
* Порог: 0.34
* WR при пороге: 50%

### Признаки

```
tf_minutes     # таймфрейм в минутах
risk_pct       # размер стопа %
reward_pct     # размер тейка %
rr_planned     # плановый RR
vol_ratio      # объём относительно среднего
hour           # час входа (UTC)
day_of_week    # день недели
is_london      # Лондонская сессия (07-12 UTC)
is_newyork     # Нью-Йорк сессия (13-18 UTC)
is_night       # ночь (22-06 UTC)
is_asia        # Азия (00-07 UTC)
dir_enc        # направление (лонг/шорт)
```

### Топ признаков по важности (535 сделок)

```
exit_progress  +0.997  (ретроспективный — не используется в фильтре)
risk_pct       -0.183  (чем меньше стоп — тем лучше)
reward_pct     -0.170
tf_minutes     -0.113  (младшие TF лучше для EMA стратегии)
vol_ratio      +0.040
is_weekend     -0.018
```
