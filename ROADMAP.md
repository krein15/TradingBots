### 3.1 Критерии перехода
```
Минимальные требования для реальных денег:

  Просадка <= 20% за последние 30 дней
  Стабильность: WR не падает 2 недели подряд
  ML ROC-AUC >= 0.70
```

### 3.2 Размер позиции (Kelly Criterion)
```python
# Сейчас: фиксированный риск 5%
# Будет: динамический размер по Kelly

def kelly_fraction(wr, rr):
    # Kelly = WR - (1-WR)/RR
    return wr - (1 - wr) / rr

# При WR=35%, RR=4:
kelly = 0.35 - 0.65/4 = 0.1875  # 18.75% от депозита
# Используем 1/4 Kelly для безопасности = ~5%

# При WR=50%, RR=4:
kelly = 0.50 - 0.50/4 = 0.375   # 37.5%
# 1/4 Kelly = ~9%
```
- [ ] Динамический расчёт размера позиции по Kelly
- [ ] Максимальный риск на сделку: 10%
- [ ] Максимальный риск на день: 20% от депозита

## Математика прибыли

### Теоретический расчёт (WR 35%, RR 1:4)
```
На 100 сделок:
  35 побед  × +4R = +140R
  65 потерь × -1R = -65R
  Чистая прибыль:  +75R = +75% депозита

При риске 2% за сделку:
  100 сделок → +150% депозита
  Минус комиссии (~20%) → +130% реально
```

### Прогноз по месяцам
```
Бот #1 делает ~350 сделок/месяц

Консервативно (риск 1%):
  $500  депозит → +$325/мес  (+65%)
  $1000 депозит → +$650/мес  (+65%)
  $5000 депозит → +$3250/мес (+65%)

Умеренно (риск 2%):
  $500  депозит → +$650/мес  (+130%)
  $1000 депозит → +$1300/мес (+130%)
  $5000 депозит → +$6500/мес (+130%)
```

### С реинвестированием (compound, риск 1%)
```
Месяц 1:  $1000 → $1650
Месяц 2:  $1650 → $2722
Месяц 3:  $2722 → $4491
Месяц 6:  → $12,200
Месяц 12: → $148,000
```

⚠️ Это теория при стабильном WR 35%.
Реальность: WR нестабилен, бывают серии лоссов.
Безопасный старт: риск 1-2% пока не будет 500+ сделок.
